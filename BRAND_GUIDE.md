## ü¶á Logo Design Specifications

### Primary Logo: "Evoworks Bat"
**Style:** Geometric, minimalist, modern

**Concept:**
- **Icon:** Stylized bat in flight with wings spread
- **Sound Waves:** Concentric circles emanating from bat (echolocation)
- **Color:** Orange (#ff6b35) bat on dark background
- **Alternate:** White outline for light backgrounds

### Logo Variations

#### 1. Full Logo (Horizontal)
```
[ü¶á BAT ICON] ECHO MARKETPLACE
- Use for: Headers, landing pages, marketing
- Aspect ratio: 4:1
```

#### 2. Icon Only (Square)
```
[ü¶á BAT WITH WAVES]
- Use for: Favicons, app icons, social media
- Aspect ratio: 1:1
- Minimum size: 32x32px
```

#### 3. Vertical Logo
```
    [ü¶á BAT]
      ECHO
  MARKETPLACE
- Use for: Mobile headers, narrow spaces
```

### Design Elements
- **Wings:** Angular, geometric (not realistic)
- **Evoworks Waves:** 2-3 concentric circles behind bat
- **Direction:** Bat facing right (forward motion)
- **Animation:** Subtle pulse on hover (echo effect)

## üéØ Design Principles

### 1. **Dark First**
- Primary theme is dark grey/black
- Light mode is secondary
- Orange provides vibrant contrast

### 2. **Evoworks Pattern**
- Use concentric circles/waves as recurring motif
- Apply to: loading states, hover effects, backgrounds
- Represents: signal propagation, network effects

### 3. **Precision & Intelligence**
- Clean lines, sharp edges
- No clutter, purposeful white space
- Every element serves a function

### 4. **Trust & Security**
- Professional, enterprise-grade aesthetic
- Subtle animations (not distracting)
- Clear hierarchy and information architecture
